## 👋 为什么选择 Gemini Voyager？

我们喜欢 Gemini，但有时候希望它能更有条理一些。

这就是我们开发 **Gemini Voyager** 的原因。它不只是一个工具，更是一个伴侣，帮助你保持 AI 对话的组织性、可访问性和高效性。无论你是需要管理数十个对话的研究人员、保存代码片段的开发者，还是只是喜欢井井有条的人，Voyager 都是为你设计的。

### 📍 时间线导航

**再也不会在长对话中迷路。**
可视化节点让你一眼看清对话结构。

- **即时跳转**：点击节点直接跳到那条消息
- **悬停预览**：鼠标悬停即可查看内容，无需滚动
- **标记重点**：长按可以给重要消息打星标（跨标签页同步！）
- **节点层级（实验性）**：右键设置节点层级（1-3级）或折叠子节点，完美适配分支或层级对话

### 📂 文件夹管理

**让你的想法井井有条。**
不再翻找混乱的历史记录列表。拖放对话到合适的文件夹。

- **两级层级**：文件夹和子文件夹，深度组织
- **轻松管理**：双击重命名，长按多选
- **跨设备同步**：导入/导出结构，保持设备间同步

### 💬 引用回复

**带着上下文回复。**
选中任何消息中的文字，会出现一个浮动的引用按钮。点击它，选中的内容会自动作为引用插入到回复框中。

- **上下文感知**：对用户和模型消息都有效
- **标准格式**：使用标准 Markdown 引用语法（`> ...`）

### 🧜‍♀️ Mermaid 图表渲染

**即时可视化图表。**
当 Gemini 输出 Mermaid 代码（流程图、时序图、甘特图等）时，Voyager 会自动将其渲染为可交互的图表。

- **切换视图**：一键切换渲染图和源代码
- **全屏模式**：点击图表进入全屏查看器，支持缩放和平移
- **自动检测**：支持所有主流 Mermaid 图表类型

### 🍌 NanoBanana 去水印

**无损去除 Gemini 图片水印。**
使用反向 alpha 混合算法，自动去除 AI 生成图片上的"Gemini"水印。

- **零质量损失**：重建原始像素，获得完美干净的图片
- **一次设置**：在设置中启用一次，以后自动生效
- **直接下载**：将原生下载按钮替换为 🍌 按钮，保存的就是去水印后的图片
- **隐私优先**：所有处理都在浏览器本地完成，100% 本地处理

> **注意**：此功能由 [journey-ad](https://github.com/journey-ad) 的 [gemini-watermark-remover](https://github.com/journey-ad/gemini-watermark-remover) 算法提供支持，该算法基于 [allenk](https://github.com/allenk) 的 [原始 C++ 实现](https://github.com/allenk/GeminiWatermarkTool)。🧡

<details>
<summary>开发环境设置</summary>

```bash
# 安装依赖（推荐使用 Bun）
bun i

# 开发模式（自动重载）
bun run dev:chrome   # Chrome 和 Chromium 浏览器
bun run dev:firefox  # Firefox
bun run dev:safari   # Safari（需要 macOS）

# 生产构建
bun run build:chrome   # Chrome
bun run build:firefox  # Firefox
bun run build:safari   # Safari
bun run build:all      # 所有浏览器
```

**Safari 开发**：查看 [safari/README.md](safari/README.md) 了解额外的构建步骤。

</details>
